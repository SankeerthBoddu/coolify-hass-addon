# Coolify Home Assistant Add-on Repository

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FSankeerthBoddu%2Fcoolify-hass-addon)

## About

This repository contains a Home Assistant add-on for [Coolify](https://coolify.io/) - an open-source & self-hostable alternative to Heroku / Netlify / Vercel.

## Add-ons

### [Coolify](./coolify)

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

Self-hosting with superpowers. Deploy your applications, databases, and services with ease.

## Installation

Adding this add-ons repository to your Home Assistant instance is straightforward:

1. Click the button above, or manually add this repository URL to your Home Assistant add-on store:
   ```
   https://github.com/SankeerthBoddu/coolify-hass-addon
   ```

2. Find the "Coolify" add-on in the store
3. Click "Install"
